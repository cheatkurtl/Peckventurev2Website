Wer wir sind Who We Are

Peckventure wird von einem leidenschaftlichen Team entwickelt, das sich auf einzigartige Weise ergänzt, um dieses Projekt zu verwirklichen. Peckventure is brought to life by a passionate team, each contributing uniquely to make this project a reality.

  • Andreas Goldbrunner - Projektleitung: Verantwortlich für die Koordination des gesamten Projekts sowie für die Website und das Community-Management. Andreas Goldbrunner - Project Manager: Responsible for overall project coordination, the website, and community management.
  • Leopold Maier - Head Developer: Leitet die Entwicklung des Spiels und der Website mit technischem Know-how und innovativen Ideen. Leopold Maier - Head Developer: Leads the development of the game and website with technical expertise and innovative ideas.
  • Lukas Hamburger - Developer: Unterstützt die Entwicklung des Spiels durch die Implementierung von Features und Fehlerbehebungen. Lukas Hamburger - Developer: Supports the game's development by implementing features and fixing bugs.
  • Jason Kalus - Server Manager: Verwaltert die Datenbank und stellt sicher, dass die Server stabil und sicher laufen. Jason Kalus - Server Manager: Manages the database and ensures the servers run smoothly and securely.
